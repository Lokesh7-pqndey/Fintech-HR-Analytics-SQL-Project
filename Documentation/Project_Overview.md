## Tools Used

- SQL
- GitHub

## Analysis Areas
- Workforce Summary
- Department Payroll Analysis
- Salary Variance Analysis
- Retention Risk Analysis

## Expected Outcome
Generate actionable HR insights to support workforce planning and compensation decisions.
